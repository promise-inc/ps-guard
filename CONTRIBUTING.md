# Contributing to @promise-inc/ps-guard

Thanks for your interest in contributing! This project is open source and we welcome contributions.

## How to report bugs

To report a bug, please first read our guide on [opening issues](https://github.com/promise-inc/ps-guard/issues).

Include as much detail as possible:

- Node.js version
- OS and version
- Chrome/Chromium version
- Steps to reproduce
- Expected vs actual behavior

## How to contribute code

To open a pull request, please first read our guide on [opening pull requests](https://github.com/promise-inc/ps-guard/pulls), which outlines our process for RFCs and pull requests.

### Getting started

```bash
git clone https://github.com/promise-inc/ps-guard.git
cd ps-guard
npm install
npm run build
```

### Guidelines

- Follow the existing code style
- Add tests for new features
- Keep commits clean with [conventional commits](https://www.conventionalcommits.org/)
- Make sure `npm run build` passes before submitting

## Code of Conduct

Be respectful and constructive. We're all here to build great tools.
